# Raspberry Potter
Use a wand to control the IoT (Internet of Things)

You'll need:
- Raspberry Pi
- NoIR Camera
- IR LED Array
- Python
- OpenCV
- Some sort of IR reflective wand (like the wands you get at Universal Studios)
- Things you want to control
